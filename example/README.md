# 开发测试说明

## example 项目

### 开发环境初始化
1. 安装依赖
```
npm install @rongcloud/imlib-uni --save
```

2. 使用 Hbuilder 打开该目录

3. 在 config/config.js 文件中 userList 里加入测试的账号

4. 将原生插件放到 nativeplugins 目录下

5. 在 manifest.json -> APP 原生插件配置 -> 本地插件中加入原生插件

6. 运行 -> 运行到手机 -> 制作自定义调试基座

7. 运行 -> 运行到手机 -> 运行基座选择 -> 选择自定义基座
